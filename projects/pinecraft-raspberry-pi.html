Installing Raspberry Pi OS

Beginner-friendly: This section guides you through installing Raspberry Pi OS using Raspberry Pi Imager.

Compatible Devices

  • Raspberry Pi 5
  • Raspberry Pi 4
  • Raspberry Pi 3
  • Raspberry Pi Zero 2 W (not fully tested)

Step-by-Step: Using Raspberry Pi Imager

  1. Open Raspberry Pi Imager
    If you don't have it, download and install it from https://www.raspberrypi.com/software/ on your computer.
  2. Choose device
    Click Choose Device and select the Raspberry Pi model you are using.
  3. Choose operating system
    Click Choose OS → Select Raspberry Pi OS (Other) → Raspberry Pi OS Lite (64-bit).
  4. Choose storage
    Click Choose Storage and select the SD card or USB drive you will use in the Raspberry Pi.
  5. OS customization settings
    • Set a hostname (e.g., Pinecraft)
    • Select your location
    • Create a username and password
    • Enter your Wi-Fi SSID and password
    • Enable SSH
    • Enable Raspberry Pi Connect
  6. Write the image
    Click Write.
    Warning: You are about to erase all data on this drive. This action is permanent and cannot be undone.
    Click I understand ‚Äî erase and write and wait for the process to complete.

Plug the drive into your Raspberry Pi

  • Make sure the Raspberry Pi is powered OFF
  • Insert the SD card or USB drive into the Raspberry Pi
  • If your Raspberry Pi has blue and black USB ports, use the blue ports (they are faster)
  • Power the Raspberry Pi ON

Connect via Raspberry Pi Connect

  1. Go to https://connect.raspberrypi.com
  2. Refresh the page until your device appears (look for the hostname you chose)
  3. Click Connect when it appears
Important Hostname Warning:
  • If you reuse a hostname that already exists (for example: Pinecraft-V2)
  • You MUST delete the old device from Raspberry Pi Connect first
  • This includes devices that are offline or no longer in use
  • If you do not delete the old device, the new one may not appear

Once you are connected via Raspberry Pi Connect, you are ready for the next steps. Further setup is covered later in this guide.

Installation Steps

Step 1: Open Raspberry Pi configuration

sudo raspi-config
  • Go to Advanced Options
  • Select Expand Filesystem
  • When prompted to reboot, select No
  • Exit raspi-config

Step 2: Update the system

sudo apt update
sudo apt upgrade

This ensures your Raspberry Pi has the latest software and security updates.

Step 3: Install Git

sudo apt install git

Git is required to download the Pinecraft installer from GitHub.

Step 4: Reboot the Raspberry Pi

sudo reboot

This will apply any changes and prepare your system for installation.

Step 5: Clone the Pinecraft repository

git clone https://github.com/Cat5TV/pinecraft.git

This command downloads the official Pinecraft installer to your Raspberry Pi.

Step 6: Enter the Pinecraft directory

cd pinecraft

This changes your working directory to the Pinecraft installer folder.

Step 7: Run the Pinecraft installer

sudo ./install
  • Click OK through the installer dialogs
  • Select any options you prefer
  • The default options are acceptable for most users

Notes

  • These steps are enough to get Pinecraft installed on your Raspberry Pi.
  • Configuration and usage will be covered later in this guide or video.
  • You can change installation options after setup if needed.
