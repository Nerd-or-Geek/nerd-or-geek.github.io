P4wnP1

A highly customizable USB attack platform for Raspberry Pi Zero/Zero W

Tutorial Raspberry Pi Offensive Security

Overview

P4wnP1 is an open-source, highly customizable USB attack platform for the Raspberry Pi Zero and Zero W. It enables HID attacks, network attacks, and more, all from a tiny, affordable device.

Prerequisites

Hardware Requirements

Raspberry Pi Zero or Zero W
microSD card (8GB+ recommended)
Micro USB cable (for data & power)
Computer (Windows, macOS, or Linux)

Installation Guide

1 Download & Flash P4wnP1 Image
  1. Download the latest P4wnP1 A.L.O.A. image from the GitHub Releases page.
  2. Use balenaEtcher or a similar tool to flash the image to your microSD card.
2 First Boot
  1. Insert the microSD card into your Pi Zero.
  2. Connect the Pi Zero to your computer using the USB data port (not the power-only port).
  3. Wait for the Pi to boot. It will appear as a network and/or HID device on your computer.
3 Initial Access
  1. Default credentials (if needed):
    Username: pi
    Password: raspberry
  2. For advanced setup, customization, and payloads, see the P4wnP1 Wiki.
4 Advanced Setup (Optional)

Refer to the P4wnP1 Wiki for payloads, customizations, and advanced features.
