P4wnP1

A highly customizable USB attack platform for Raspberry Pi Zero/Zero W

Tutorial Raspberry Pi Offensive Security

Overview

P4wnP1 is an open-source, highly customizable USB attack platform for the Raspberry Pi Zero and Zero W. It enables HID attacks, network attacks, and more, all from a tiny, affordable device.

Prerequisites

Verify you have all necessary hardware before starting installation.

Required Hardware

  • Raspberry Pi Zero or Zero W
  • MicroSD card (8GB+ recommended)
  • Micro USB cable (data & power capable)
  • Computer (Windows, macOS, or Linux)

Optional Hardware

  • USB OTG adapter for additional devices
  • Compact case for portability

Installation Guide

Follow these steps to set up P4wnP1 on your Raspberry Pi Zero.

Download & Flash

Download Image

Get the latest P4wnP1 A.L.O.A. image from GitHub.

Download from Releases

Flash to SD Card

Use balenaEtcher to write the image to your microSD card.

Get balenaEtcher

Insert & Connect

Insert the SD card into Pi Zero. Connect via USB data port (not power-only).

Initial Setup

Wait for Boot

The Pi will boot and appear as a network/HID device on your computer.

Default Credentials

Username: pi
Password: raspberry

Access Web UI

Open browser and navigate to the P4wnP1 web interface for configuration.
