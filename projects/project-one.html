Pinecraft

A comprehensive guide to installing and running a Minecraft server on Raspberry Pi

Tutorial Raspberry Pi Minecraft

Overview

Pinecraft is a Raspberry Pi-hosted Minecraft server setup guide. This documentation provides a simple, reliable installation process suitable for Raspberry Pi 4 (4GB+ recommended) running Raspberry Pi OS 64-bit.

Estimated Time: 30-45 minutes including downloads

Prerequisites

Before you begin, ensure you have the following:

Raspberry Pi 4 (4GB or 8GB RAM recommended)
Raspberry Pi OS (64-bit), updated to latest version
Stable power supply and cooling (heatsink recommended)
Ethernet connection (preferred over WiFi)
Router access for port forwarding (if hosting publicly)

Installation Guide

Follow these steps in order to set up your Pinecraft server.

1 Update System Packages

First, update all system packages to their latest versions:

bash 
sudo apt update
sudo apt full-upgrade -y
sudo reboot
Note: The system will reboot after the upgrade. This is normal and expected.

2 Install Java Runtime

Minecraft servers require Java. Install OpenJDK 17 (recommended for modern servers):

bash 
sudo apt install -y openjdk-17-jre-headless

Verify installation:

bash 
java -version

3 Create Dedicated User & Folders

Create a dedicated system user and directories for the server:

bash 
sudo adduser --system --home /opt/pinecraft --group minecraft
sudo mkdir -p /opt/pinecraft
sudo chown -R minecraft:minecraft /opt/pinecraft
Running the server as a dedicated user improves security and isolation.

4 Download & Configure Server

Download a server jar (this example uses PaperMC, a high-performance Minecraft server):

bash 
sudo -u minecraft bash -c '
cd /opt/pinecraft
# Visit https://papermc.io/downloads to get the latest build
# Replace  with the actual download URL
wget -O server.jar 
java -Xms1G -Xmx2G -jar server.jar nogui
'

Accept the EULA:

bash 
sudo sed -i "s/eula=false/eula=true/" /opt/pinecraft/eula.txt
Memory Configuration: Adjust -Xms and -Xmx based on your Pi's RAM:
  • 4GB Pi: -Xms512M -Xmx1536M
  • 8GB Pi: -Xms1G -Xmx3G

5 Create Systemd Service

Create a systemd service to run the server on boot:

bash 
sudo tee /etc/systemd/system/pinecraft.service > /dev/null <<'SERVICE'
[Unit]
Description=Pinecraft Minecraft Server
After=network.target

[Service]
Type=simple
User=minecraft
Group=minecraft
WorkingDirectory=/opt/pinecraft
ExecStart=/usr/bin/java -Xms1G -Xmx2G -jar /opt/pinecraft/server.jar nogui
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
SERVICE

Enable and start the service:

bash 
sudo systemctl daemon-reload
sudo systemctl enable pinecraft
sudo systemctl start pinecraft

Check server status:

bash 
sudo systemctl status pinecraft

6 Open Port & Test Connection

To access your server remotely, configure port forwarding on your router:

  1. Log into your router's admin panel
  2. Find Port Forwarding settings
  3. Forward TCP port 25565 to your Pi's local IP address
  4. Note your external IP (use curl ifconfig.me on the Pi)

Connect from Minecraft Java Edition:

  1. Launch Minecraft
  2. Go to Multiplayer â†’ Direct Connect
  3. Enter your Pi's IP (local: 192.168.x.x or external IP)
  4. Click Join Server

Maintenance & Management

Stop Server

Safely stop the server:

sudo systemctl stop pinecraft

Start Server

Start the server:

sudo systemctl start pinecraft

Restart Server

Restart the service:

sudo systemctl restart pinecraft

Backup Server

Backup your world and configs:

sudo tar -czf backup.tar.gz /opt/pinecraft/

Troubleshooting

Server won't start

Check the logs:

sudo journalctl -u pinecraft -n 50

Common issues: insufficient memory, Java not installed, or EULA not accepted.

Can't connect from outside network

Verify port forwarding is configured correctly. Test with:

curl ifconfig.me:25565 -v

Server is slow or lagging

Check system resources:

top

Reduce player count or lower world render distance in server.properties.
